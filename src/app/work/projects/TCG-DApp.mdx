## Overview

The Trading Card Game (TCG) DApp is a blockchain-based gaming platform that allows users to collect, trade, and battle using NFT hero cards. It includes a full-featured marketplace and a custom ERC20 utility token (`TCGToken`).

## Stack

* **Frontend**: Next.js (App Router), Tailwind CSS, Once UI
* **Blockchain**: Solidity, Hardhat
* **Wallets**: RainbowKit, Wagmi v2, Ethers.js
* **State**: Zustand, React Query

## Features

* Buy/sell NFTs on the integrated Marketplace
* Approve & transfer ERC20 tokens for purchases
* View metadata from `tokenURI`
* Responsive layout with dark mode

## Status

Core marketplace features are complete. Game mechanics like turn-based battles and sound effects are in progress.
