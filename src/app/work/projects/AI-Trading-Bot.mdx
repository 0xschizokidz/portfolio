## Overview

This bot retrains every 15 minutes using the latest OHLCV data and sends live Telegram alerts for high-confidence buy signals.

## Stack

* **Libraries**: pandas, XGBoost, scikit-learn, ccxt
* **ML Labels**: Based on 2% TP / 1% SL exits
* **Execution**: Live predictions with confidence threshold

## Features

* Retrains on rolling data
* Real-time prediction and alerting
* Label-based supervised learning

## Status

Working version deployed. Improvements underway for classification confidence and risk-adjusted filters.
