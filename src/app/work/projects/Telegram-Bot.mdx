## Overview

This is a customizable, open-source LLM assistant running locally using Ollama and integrated with n8n for workflow automation.

## Stack

* **LLMs**: Mistral, Phi, LLaMA2 (via Ollama)
* **Automation**: n8n
* **Memory**: Redis
* **Platform**: Docker
* **Messaging**: Telegram bot

## Features

* /commands-based interactions
* Persistent memory with Redis
* Dynamic LLM model switching
* Secure, offline AI pipeline

## Status

Live and extensible. Next steps include integrating trading signal notifications and webhook triggers.
